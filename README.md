# 📊 Customer Churn Analysis, Segmentation, and Prediction using Machine Learning

## 📌 Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. Retaining existing customers is often more cost-effective than acquiring new ones. This project analyzes customer behavior using a telecom customer dataset to identify the factors influencing customer churn and develops a machine learning model to predict customers who are likely to leave.

The project follows a complete Data Analytics workflow, including data preprocessing, exploratory data analysis (EDA), customer segmentation, churn prediction, customer retention strategies, and advanced visualizations.

---

## 🎯 Project Objectives

- Clean and preprocess the customer dataset.
- Perform Exploratory Data Analysis (EDA) to discover customer behavior.
- Segment customers based on tenure, monthly charges, contract type, and customer lifetime value.
- Build a Machine Learning model to predict customer churn.
- Identify high-risk customers.
- Recommend data-driven customer retention strategies.

# 📈 Key Insights

- Approximately 27% of customers have churned.
- Customers with month-to-month contracts exhibit the highest churn.
- Fiber Optic users churn more frequently.
- Customers with shorter tenure are more likely to leave.
- Higher monthly charges are associated with increased churn.
- Electronic Check users have a higher churn rate.
- Long-term customers are generally more loyal.

---

# 💡 Business Recommendations

- Encourage customers to switch to long-term contracts.
- Launch onboarding programs for new customers.
- Offer personalized discounts for high-risk customers.
- Improve Fiber Optic service quality.
- Promote automatic payment methods.
- Provide loyalty rewards to long-term customers.
